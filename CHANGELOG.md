# Changelog

All notable changes to Lean PM Skills are documented here. This project follows [Semantic Versioning](https://semver.org/) — new skills and features bump minor, fixes and improvements bump patch.

---

## [1.2.1] — 2026-04-26

### Added
- **Story Breakdown** skill — decompose large problems, epics, or initiatives into independently shippable slices using splitting patterns

### Changed
- Skills count updated to 23

---

## [1.2.0] — 2026-04-26

### Added
- **PRD Writing** skill — problem statement to builder-ready PRD in 1-2 hours
- **Epic Definition** skill — structure epics with strategic context, feature breakdown, milestones
- **Feature Specification** skill — detail features with requirements, edge cases, integration points
- **Story Writing** skill — developer-ready stories with testable Given/When/Then acceptance criteria
- **Analytics & Insights** skill — deep product data investigation beyond routine dashboards
- New "Definition & Documentation" skill category in README

### Changed
- Version bumped to 1.2.0
- Skills count updated to 22

---

## [1.1.0] — 2026-04-26

### Added
- **Experiment Design & Monitoring** skill — lean build-measure-learn loop from hypothesis to validated learning
- **Roadmap Planning** skill — translate strategy and OKRs into sequenced, audience-specific roadmaps
- Framework decision flowchart in FRAMEWORKS.md for choosing between similar frameworks
- CHANGELOG.md for version tracking
---

## [1.0.0] — 2026-04-26

### Added
- Initial release with 15 skills: Discovery Process, Stakeholder Alignment, Pricing & Packaging, Growth & Retention Diagnostics, Product Strategy & Vision, User Research Methods, Go-to-Market Planning, Tech Debt Communication, Sprint & Release Planning, Metrics Review, Competitive Response, OKR & Goal Setting, Feature Sunset & EOL, Post-Mortem, Build vs. Buy
- Core behavioral guidelines (CLAUDE.md)
- PM frameworks adapted for AI-native speed (FRAMEWORKS.md)
- 15 artifact templates (TEMPLATES.md)
- Agentic workflows guide (AGENTIC-WORKFLOWS.md)
- AI-native workflow guide (AI-NATIVE-WORKFLOW.md)
- Getting started guide (GETTING-STARTED.md)
- Before/after examples (EXAMPLES.md)
- Cursor IDE integration (.cursor/rules)
- Claude Code plugin packaging (.claude-plugin/plugin.json)
